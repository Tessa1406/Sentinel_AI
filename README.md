# Sentinel_AI
Sentinel AI is a college mini project that simulates a cyber security dashboard. It uses a React + Vite frontend and an Express backend to create a security operations console-style interface.

# Features
Login portal with email and password placeholders Dashboard with monitored nodes, status, metrics, and zone coverage Threat view with vulnerability scenarios and patch simulation Logs view with active alerts and event search Assistant chat interface for simulated AI security responses Settings view with operator email update and reset option

# Tech Stack
React + Vite Express
backend TypeScript 
dotenv for environment configuration
Browser localStorage for demo session state
